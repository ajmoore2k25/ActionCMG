
# ActionCMG Site

A minimal Next.js + Tailwind one‑page site for Action.

## Quick start
1. Install: `npm install`
2. Run dev: `npm run dev`
3. Build: `npm run build`

### Deploy on Vercel
Import the repo in Vercel. Add your custom domain in **Settings › Domains**.

### Configure
- Replace Formspree endpoint in `app/page.tsx` (`https://formspree.io/f/your-form-id`).
- Add a real `public/og-image.png` (1200×630).
- Update metadata in `app/layout.tsx` if using a different domain.
